# Monthly Update File
| Field                 | Description                                                              | Format                                              | Todo/Notes                                                                                                                                                                                               |
| --------------------- | ------------------------------------------------------------------------ | --------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CompanyName           | The entity creating the file                                             | Always "Ashford Borough Council"                    |                                                                                                                                                                                                          |
| CompanyOPCSRef        | An identifier for the company named above                                | Always "29UB"                                       |                                                                                                                                                                                                          |
| DistrictRef           | An identifier of the elector's nearest polling station                   | Matches values from Dec col "Elector Number Prefix" |                                                                                                                                                                                                          |
| StreetName            | The street the address is on (sans number)                               | **Optional**<br>- str                               |                                                                                                                                                                                                          |
| PropertyAddress1      | The address of the elector.                                              | **Optional**<br>- str                               | 1-5 map to address 1-5 of the annual file. If address had a line 6, that's missed off, but seems to be usually the postcode, which can be found in the PropertyPostCode column                           |
| PropertyAddress2      | The address of the elector.                                              | **Optional**<br>- str                               | A few records don't have an address mapped to them.                                                                                                                                                      |
| PropertyAddress3      | The address of the elector.                                              | **Optional**<br>- str                               |                                                                                                                                                                                                          |
| PropertyAddress4      | The address of the elector.                                              | **Optional**<br>- str                               |                                                                                                                                                                                                          |
| PropertyAddress5      | The address of the elector.                                              | **Optional**<br>- str                               |                                                                                                                                                                                                          |
| PropertyPostCode      | The postcode of the elector                                              | **Optional**<br>- str                               |                                                                                                                                                                                                          |
| PropertyAddressPrefix | Looks to be the first line of the address where it's a flat or something | **Optional**<br>- str                               | Maps exactly to PropertyAddress1 apart from double spaces added for some reason                                                                                                                          |
| ElectorNumber         | The number of the house                                                  | Optional<br>- str (to account for 12a flats)        | An inconsistent record of the number - some have it missing but can be found from the address "e.g. 3 Bath Mews"<br>Some have it missing but understandably, as the house is named rather than numbered. |
| ElectorForename       | The forename of the elector                                              | **Optional**<br>- str                               | Some records are missing a name                                                                                                                                                                          |
| ElectorSurname        | The surname of the elector                                               | **Optional**<br>- str                               | Some records are missing a name                                                                                                                                                                          |
| ElectorDOB            | The date of birth for electors that are approaching the majority age     | **Optional**<br>DD/MM/YY<br>                        |                                                                                                                                                                                                          |
| ElectorCreatedMonth   | Flags if the elector is being created                                    | int<br>1 if being created<br>0 for not              | Looks to flag new electors at a given address. all of these have a /1 or greater                                                                                                                         |
| ElectorChangedMonth   | Flags if the elector record is being changed                             | int<br>1 if being edited, 0 for not                 | *Seems* to be for name changes etc, rather than people moving                                                                                                                                            |
| ElectorDeletedMonth   | Flags if the elector record is being deleted                             | int<br>1 if being deleted, 0 for not                | For people leaving the address. If they are moving in the borough, then they get 2 entries, a deletion from their old address and a creation in their new one                                            |
| MarkersRegisterText   | Looks to be the same flags as in Elector Markers                         | **Optional**<br>- B, F, L (enum)                    |                                                                                                                                                                                                          |
| RecordCount           | The row number in the file                                               | int (sequential)                                    |                                                                                                                                                                                                          |
| RecordTotal           | The total number of rows in the file                                     | int (same for each row)                             |                                                                                                                                                                                                          |
| CurrentRegisterMonth  | The month the file was produced                                          | str                                                 |                                                                                                                                                                                                          |
